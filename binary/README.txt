The "binary" folder contains the files of the prototype for distribution:

- csmcc16.jar: Jar files for the compiled classes of the prototype
- csmcc16.properties: properties file for the prototype
- javafx-sdk-11.0.2/: JavaFX distribution Library
- README.txt: this file
- startApp.sh: start up script for the prototype (for Linux & MacOS)
- startApp_hc.sh: start up script of the "High Contrast" UI version for the prototype (for Linux & MacOS)
- startApp.bat: start up script for the prototype (for Windows)
- startApp_hc.bat: start up script of the "High Contrast" UI version for the prototype (for Windows)

Note: To run the prototype, you should have JDK 11+ and JavaFX 11+ installed.
